# Chess Swiss Tournament Web Service

Веб‑сервис для проведения шахматных турниров по швейцарской системе.  
Проект предназначен для дипломной/курсовой работы по специальности  
**09.02.07 «Информационные системы и программирование»** и демонстрирует:

- разработку веб‑приложения на современном фреймворке;
- проектирование базы данных и бизнес‑логики;
- реализацию алгоритма швейцарской системы;
- создание удобного пользовательского интерфейса.

## Стек технологий

- **Backend**: Python 3 + FastAPI
- **База данных**: SQLite (через SQLAlchemy, при необходимости можно перейти на PostgreSQL)
- **Шаблоны**: Jinja2
- **Frontend**: HTML5, CSS3, Bootstrap 5, немного JavaScript
- **Аутентификация**: своя реализация (cookie‑сессия + хэш пароля)

## Возможности приложения

- **Создание турниров** администратором:
  - название, количество туров;
  - контроль времени (например, 10+5);
  - лимит участников.
- **Регистрация участников** на турнир (ФИО/ник, рейтинг Elo опционально).
- **Жеребьёвка по швейцарской системе**:
  - сортировка по очкам (score) и рейтингу;
  - по возможности участники не встречаются дважды;
  - обработка нечётного количества игроков (bye с 1 очком);
  - учёт цветового баланса (примерное чередование белые/чёрные).
- **Управление турами**:
  - генерация следующего тура;
  - страница «Текущий тур» со списком пар;
  - ввод результатов партий (1‑0, 0‑1, 0.5‑0.5).
- **Турнирная таблица**:
  - очки;
  - Buchholz;
  - Median‑Buchholz;
  - баланс цвета.
- **История туров и партий** для каждого турнира.
- **Экспорт турнирной таблицы в CSV**.
- **Просмотр партии**:
  - отдельная страница партии;
  - хранение и редактирование PGN / текстовой нотации.
